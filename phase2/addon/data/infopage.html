Firefox Guide

Firefox Guide is a tool that proactively recommends features or extensions that are likely to be useful to you based on how you use Firefox. This tool is experimental and will be automatically removed from your Firefox after a period of time.

The picture below shows different parts of a recommendation notification.

  • 1

    Hides the notification

  • 2

    Let's you tell us why you don't like this notification

  • 3

    Shows more information about Firefox Guide

  • 4

    Gives a brief description of this feature

  • 5

    Briefly tells you why it thinks you might like this feature

  • 6

    Let's you install or try it right from the notification panel!

  • 7

    Click to learn more about the feature

Can I remove this tool?

Note that Firefox Guide is a temporary tool and will be automatically removed after a period of time. If you still want to disable it, click here to see the instructions.

Removing Firefox Guide
  1. Open the Add-ons page
  2. Find "Firefox Guide Study 1" and click the "Remove" button
